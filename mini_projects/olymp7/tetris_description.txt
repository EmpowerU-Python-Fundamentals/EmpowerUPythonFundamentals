
Тетріс — класична аркадна гра, реалізована на Python з використанням Pygame.

Функціонал для гравця:
- Фігури різних форм падають зверху вниз.
- Мета — скласти цілі горизонтальні лінії без прогалин.
- Заповнені лінії зникають, приносячи очки.
- Гра поступово прискорюється із підвищенням рівня.
- Якщо фігури досягають верху — гра закінчується.

Керування:
- Стрілки ← → — рухати фігуру вліво/вправо.
- Стрілка ↓ — прискорити падіння.
- Стрілка ↑ або X — обертати фігуру.
- Z — обертати в інший бік.
- Пробіл — миттєво опустити фігуру (hard drop).
- C — зберегти фігуру в "HOLD".
- P — пауза гри.
- ESC — вихід з гри.

Мета гри: набрати якомога більше очок, очищаючи лінії.
